
Before you can run this application, please ensure that you have the following dependencies installed on your system:

1. [Node.js](https://nodejs.org/)
   - To check if Node.js is installed, open a terminal and run the following command:
     ```sh
     node -v
     ```
   - If Node.js is not installed or the version is outdated, please visit the [Node.js website](https://nodejs.org/) and follow the installation instructions for your operating system.

2. [TypeScript](https://www.typescriptlang.org/)
   - To check if TypeScript is installed, open a terminal and run the following command:
     ```sh
     tsc -v
     ```
   - If TypeScript is not installed, you can install it globally by running the following command:
     ```sh
     npm install -g typescript
     ```

3. [Git](https://git-scm.com/)
   - To check if Git is installed, open a terminal and run the following command:
     ```sh
     git --version
     ```
   - If Git is not installed, please visit the [Git website](https://git-scm.com/) and follow the installation instructions for your operating system.

Make sure that these dependencies are installed and configured correctly before proceeding with the setup and execution of this application.

Once you have verified and installed the required dependencies, you are ready to run this project.
